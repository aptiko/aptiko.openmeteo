aptiko.openmeteo Ansible collection
===================================

This collection currently only contains the ``enhydris-instance`` role, which
helps install Enhydris.

Installation
------------

::

    ansible-galaxy collection install git+https://github.com/aptiko/aptiko.openmeteo.git
